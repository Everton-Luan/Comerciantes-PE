# Comerciantes PE

O Comerciantes PE é um marketplace simples para divulgação de produtos de comerciantes.
O objetivo deste projeto é proporcionar uma experiência próxima à de um projeto 
real, permitindo praticar desenvolvimento Full Stack, integração entre Front-end e Back-end, trabalho
em equipe e utilização de boas práticas de desenvolvimento.
Todo o desenvolvimento será acompanhado pelo Gerente do Projeto, responsável por orientar,
esclarecer dúvidas, validar as entregas e realizar as revisões do sistema.
